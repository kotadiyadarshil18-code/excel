# 📊 Excel Sales Data Analysis Project

## 📌 Project Overview

This project analyzes customer transaction data using Microsoft Excel.
The goal is to extract insights, identify trends, and create an interactive dashboard.

---

## 📂 Dataset Information

* File Name: Final pr.xlsx
* Total Records: 200+ transactions
* Fields Included:

  * Transaction ID
  * Date
  * Customer ID & Name
  * Product Details
  * Category
  * Quantity & Price
  * Payment Method
  * Region
  * Customer Segment
  * Total Amount

---

## 🎯 Objectives

* Analyze sales performance
* Identify top customers
* Find best-selling products
* Compare sales across regions
* Build a visual dashboard

---

## 🛠️ Tools & Features Used

* Excel Functions:

  * SUM, AVERAGE
  * XLOOKUP / INDEX-MATCH
  * FILTER
  * IF
  * DATEDIF, TODAY
* Data Analysis Toolpak
* Pivot Tables
* Charts (Bar, Line, Pie)
* Conditional Formatting
* What-If Analysis (Goal Seek / Scenario Manager)
